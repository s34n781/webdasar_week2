HTML Basic

Code Structure

        <!DOCTYPE html>
        <html>
            <head>
                <title> Document </title>
                <link rel="stylesheet" href="style.css">
            </head>
            <body>
                ...
            </body>
        </html>

Tag HTML

< DIV >

div adalah elemen pembungus yang digunakan untuk mengelompokkan elemen-elemen lain. Tag ini sering digunakan untuk keperluan tata letak dan styling dengan CSS.

< SPAN >

span mirip dengan div, tetapi digunakan untuk mengelompokkan teks dalam satu baris. Ini berguna untuk menerapkan gaya tertentu pada bagian kecil dari teks tanpa mempengaruhi seluruh paragraf.

LIST

ul dan ol digunakan untuk membuat daftar. ul digunakan untuk daftar yang tidak berurutan (unordered list), sementara ol digunakan untuk daftar yang berurutan (ordered list).
